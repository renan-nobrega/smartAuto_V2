Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside driveon light, dark or rtl folder.

Pages

    • index-box.html for Home page Box version
    • about.html is for About us page
    • contact.html is for Contact page
    • blog.html is for Blog page
    • blog-details.html is for blog details page
    • gallery.html is for 2 Column Gallery page
    • hero-slider.html for Hero slider version
    • hero-kenburne.html for Hero kenburne version
    • hero-parallax.html for Hero parallax version
    • hero-slideshow.html for Hero slideshow version
    • hero-text.html for Hero text effect version
